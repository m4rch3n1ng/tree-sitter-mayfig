
<h1 align="center">tree-sitter-mayfig</h1>
<p align="center">a tree-sitter grammar for the <a href="https://github.com/m4rch3n1ng/mayfig">mayfig config language</a></p>

you can find queries for both the helix editor and for neovim in the
[queries directory](https://github.com/m4rch3n1ng/tree-sitter-mayfig/tree/main/queries).

to get them to work for other editors you might have to adjust them.
