
<h1 align="center">tree-sitter-mayfig</h1>
<p align="center">a tree-sitter grammar for the <a href="https://github.com/m4rch3n1ng/mayfig">mayfig config language</a></p>

the available queries in the [queries directory](https://github.com/m4rch3n1ng/tree-sitter-mayfig/tree/main/queries)
are intended for the [helix editor](https://helix-editor.com). to get them to work for other editors
you might have to adjust them.
